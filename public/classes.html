Relevant Coursework

Junior Year

  • Communication networks - learned the basics of socket programming, learned about the history of computer networks
  • Combinatorics - learned how to count
  • Operating systems - learned the internal workings of the Linux operating system (in progress)
  • Databases - learned the basics of database creation, made a database and frontend to track friendly competition (in progress)
  • Computer architecture 2 - Learned about advanced RISC pipelining (in progress)

Sophomore Year

  • Object-oriented software development - learned the fundamentals of object-oriented programming using Java, created a joust-like game.
  • AC circuits - learned the basics of AC circuit analysis with a touch of power
  • Matrix Algebra and Differential equations 1 and 2- learned matrix algebra, first order differential equations, and later higher-order differential equations
  • Data structures and algorithms - learned how various data structures are implemented in Java, created a self-balancing binary tree
  • Introduction to digital systems - learned about state machines, used an fpga to create a rudimentary pong game

  • Continuous-Time Signals and systems - analyzed continuous signals using Fourier transforms
  • Introduction to probability - learned how to calculate both discrete and continuous probabilities

Freshman Year
